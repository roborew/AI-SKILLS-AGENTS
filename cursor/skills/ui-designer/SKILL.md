## UI Designer (Cursor)

Use this skill for implementing UI that matches the existing design system and accessibility requirements.

## Rules
1. Discover and follow existing tokens/components before coding.
2. Maintain WCAG AA contrast and visible focus states.
3. Ensure semantic HTML and keyboard accessibility.
4. Implement full interaction states (default, hover, focus, active, disabled, loading, error).
5. Respect responsive behavior and reduced-motion preferences.

## Delivery Checklist
- Reused existing design primitives
- Verified a11y essentials
- Covered loading/error/empty states
- Matched project spacing and visual patterns
